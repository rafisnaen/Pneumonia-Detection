Versi - versi penting yang digunakan :

Python : 3.11.9 atau 3.11 lainnya
TensorFlow : 2.14.0
NumPy  :1.26.4
Pandas : 2.2.3
scikit-learn : 1.6.0
*Jika menginstal tensorflow 2.14.0 maka keras juga akan terinstal dengan versi yang sama dengan tensorflow. Program akan error pada library version jika tensorflow yang terinstal adalah versi terbaru.

*Jika di download di binusmaya atau di github, file code lain akan error Ketika dijalankan (Kecuali predict) karena file code selain predict, terhubung dengan dataset yang disimpan di local. Untuk mencoba kode selain predict, download dataset chest_xray pada link drive yang tersedia di laporan, taruh dataset di direktori yang sama dengan kode - kode.

*Jika di download di binusmaya atau di github, Kami menyediakan beberapa dataset untuk dicoba diprediksi menggunakan kode predict. Dataset berjumlah 5856 gambar, yang tidak memungkinkan untuk diupload ke binusmaya.

*Jika di download di google drive, langsung download satu folder PKM-KC_AIProject_Tester dan simpan di penyimpanan local computer.

Saya juga sertakan folder virtual environtment yang kami pakai di link drive. Model yang kami sediakan adalah model dengan hasil latih terbaik kami, dengan test accuracy 90% memakai file train yang sama. ( dapat dilihat dengan menjalankan evaluate ).
